# Simulador de Cinemática em Python

Projeto desenvolvido em Python para simulação de Movimento Retilíneo Uniforme (MRU) e Movimento Retilíneo Uniformemente Variado (MRUV), com geração automática de gráficos físicos.

## Objetivo

Aplicar conceitos de física e programação científica para modelar movimentos da cinemática clássica utilizando computação e visualização gráfica.

## Funcionalidades

* Simulação de MRU
* Simulação de MRUV
* Geração automática de gráficos
* Salvamento de imagens
* Organização modular do código

## Tecnologias utilizadas

* Python
* NumPy
* Matplotlib

## Estrutura do projeto

```bash
assets/
docs/
graficos/
simulacoes/
main.py
```

## Conceitos físicos aplicados

### MRU

Equação:
s = s0 + vt

### MRUV

Equação:
s = v0t + (1/2)at²

## Como executar

```bash
pip install numpy matplotlib
python main.py
```
